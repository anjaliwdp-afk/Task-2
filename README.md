Task 2 – Basic CRUD Application
Internship: ApexPlanet Software Pvt Ltd

Objective:
The objective of this task is to develop a simple web application that performs CRUD (Create, Read, Update, Delete) operations and implements basic user authentication.

Technologies Used:
PHP
MySQL
phpMyAdmin
XAMPP
HTML
CSS

Database Setup:
A MySQL database named "blog" was created.
Two tables were implemented:

1. users
   - id
   - username
   - password

2. posts
   - id
   - title
   - content
   - created_at

Features Implemented:
• User registration system
• User login system
• Password hashing for security
• Session management for authenticated users
• Create new posts
• Display all posts
• Edit existing posts
• Delete posts

Project Files:
db.php – database connection
register.php – user registration
login.php – user login
logout.php – logout functionality
dashboard.php – display posts
add_post.php – add new post
edit_post.php – update post
delete_post.php – delete post
style.css – basic user interface styling

Learning Outcomes:
Through this task I learned how to connect PHP with MySQL, implement CRUD operations, handle user authentication using sessions, and structure a basic web application.

Conclusion:
The application successfully performs CRUD operations and demonstrates basic backend development using PHP and MySQL.
